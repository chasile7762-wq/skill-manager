# Codex Skill 管理器打包交付说明

## 1. 目标产物

本项目的 Windows 桌面产物目标为：

- `.exe` 可执行文件
- 或 Tauri 默认输出的 Windows 安装包 / 便携版包

## 2. 打包前提

打包 `.exe` 前，本机必须具备：

- Node.js
- Rust toolchain
- Tauri CLI
- Windows C++ 编译环境

## 3. 当前项目已完成的内容

- 汉化界面
- 前端构建验证
- Tauri 配置接入
- 安装、扫描、删除、项目发现逻辑
- 用户说明手册与交付说明文档

## 4. 打包命令

```powershell
npm install
npm run build
npm run desktop:build
```

或直接使用一键脚本：

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\build-desktop.ps1
```

## 5. 默认输出位置

打包成功后，请到以下目录查找产物：

```text
src-tauri/target/release/bundle/
```

## 6. 交付建议

建议最终一起交付以下内容：

- 打包好的 `.exe` 或安装包
- `用户使用手册.md`
- `打包交付说明.md`
- 源代码仓库

## 7. 当前机器状态说明

如果当前机器缺少 Rust 或 Windows 编译工具链，则无法直接产出 `.exe`，需要先补齐环境后再执行打包。

当前这台机器的实际阻塞点是：

- `cargo` 没有完成默认工具链安装
- 因此 `npm run desktop:build` 报错：`failed to get cargo metadata: program not found`
